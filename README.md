# 🤖 Prompt Engineering Templates Library

A collection of reusable **ChatGPT prompt templates** 
for HR, Data Analysis, and Business Reporting tasks.

---

## 📌 Overview

This library contains ready-to-use prompt templates 
designed to save time and improve consistency when 
using AI tools like ChatGPT for business tasks.

---

## 📂 Templates Available

| Template | Description | Use Case |
|---|---|---|
| 🗂️ JD Formatter | Formats raw job descriptions | HR & Recruitment |
| 📧 Email Writer | Writes professional emails | Business Communication |
| 📊 Data Analyst | Analyzes data & finds insights | Data Analysis |
| 📝 Report Summarizer | Summarizes reports to executive summary | Reporting |
| 📈 KPI Analyzer | Analyzes KPI performance | Performance Management |

---

## 🚀 How to Use

1. Open the template file you need
2. Copy the prompt
3. Replace the `[PLACEHOLDERS]` with your content
4. Paste into ChatGPT
5. Get your formatted output instantly!
